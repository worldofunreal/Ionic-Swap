## **NFT marketplace collection page**

### 🖥️ **Desktop Layout**

#### **Banner Section**

- **Left-aligned collection avatar & info**:
  - Collection name (e.g., `X FIGURES`)
  - Verified icon
  - Creator name with link
  - Blockchain (Ethereum), Total items (404), Launch date (Jun 2025), Category (Art), Mint status

- **Right-aligned stats**:
  - Floor Price (e.g., `0.0792 ETH`)
  - Top Offer (e.g., `0.0365 WETH`)
  - Total Volume (e.g., `231.69 ETH`)
  - 1D Floor % change
  - 24H volume
  - Number of owners (with %)

#### **Tab Navigation**

- Tabs: `Explore`, `Items`, `Offers`, `Holders`, `Traits`, `Activity`, `About`

#### **Items Filtering Sidebar (Left)**

- **Status** (All / Listed)
- **Rarity**
- **Price**
- **Marketplaces**
- **Traits**:
  - Edition Size
  - Edition Type
  - Platform
  - Rank
  - Release Year
  - Series

#### **Item Listing Area (Right)**

- Top controls:
  - Search by item or trait
  - Sort dropdown (`Price low to high`, etc.)
  - View toggle (grid/list)
  - Insights button

- Grid of NFT cards:
  - Image
  - Name (e.g., `WANT_3`)
  - Price (e.g., `0.0792 ETH`)
  - Last sale or time left (if auctioned)
  - Token ID

#### **Buy Area (Bottom Bar)**

- Quantity selector
- Max price input
- Buttons: `Buy floor`, `Make collection offer`

---
